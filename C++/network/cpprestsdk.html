cpprestsdk

標準の c++ には HttpClient や REST、libcurl のようなインターネット通信を行う仕組みがありません。とりあえず何かしらの外部ライブラリを使用する必要があります。

本ページでは、Microsoft が OSS として公開している cpprestsdk を試してみて、動作確認できたコードなどを記録に残していきたいと思います。
