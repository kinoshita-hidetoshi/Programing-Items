1. 一様分布 (std::uniform_int_distribution)

 

[概要]

一様分布の例をプログラム作成して動かしてみます。std::uniform_int_distribution を使用します。

 

[環境]

コンパイラ : Visual Studio 2012  
OS : Windows8 64bit 日本語版  

 

サンプルプログラム

#include <random>
#include <iostream>

int main()
{
    std::random_device rd;                        // シードを作成
    std::mt19937 gen(rd());                       // 乱数エンジンの宣言と初期化
    std::uniform_int_distribution<> dis(1, 10);   // 1～10の範囲のintを一様分布で生成 
    for(int n=0; n<10; ++n){
        std::cout << dis(gen) << ' ';             // 乱数生成
    }
    std::cout << '\n';
}

 

出力結果

 

上記プログラムで使用している以下の関数は全て C++11 の規格です。このため Visual Studio 2012 以降の環境を必要とします。

  • std::random_device
  • std::mt19937
  • std::uniform_int_distribution
