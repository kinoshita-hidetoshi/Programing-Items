Modern C++ 入門

21世紀に入ってまったく別物と言えるプログラミング言語に進化した C++。"Modern C++" と称される C++ について、Modern C++ らしい言語仕様を紹介している記事をこちらにメモしていきます。

たとえばインスタンスの生成に new を、破棄に delete をというのが常道でしたが、もはやそれらは非推奨なのです。このようなかつては常識だったことが非推奨になっているという状況を学んでいければと思います。
