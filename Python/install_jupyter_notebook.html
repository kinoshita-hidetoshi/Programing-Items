1. Jupyter Notebook をインストールする方法

[概要]

Python がインストール済みの環境であれば、Jupyter Notebook のみを追加でインストールすることができます。
以下のコマンドを入力することで行います。

 

python -m pip install --upgrade pip
python -m pip install jupyter

 

 

 

2. Jupyter Notebook を起動する

コンソールから jupyter notebook + [Enter] をインプットすることで起動します。

 

こんな感じで起動しました。
