1. パッケージのインストール

最も基本のパッケージインストールの方法です。

$ pip install <package-name>

 

 

2. パッケージのアンインストール

パッケージをアンインストールします。

$ pip uninstall <package-name>

 

 

3. パッケージのバージョン指定インストール

 

$ pip install <package-name>==<version>

(具体例) 

$ pip install django==2.1.2

 

 

4. pip 依存関係の確認

pip 依存関係の確認を行います。

$ pip check

 

問題ない場合、下記のように出力します。

No broken requirements found.

依存関係に問題ある場合、下記例のように出力します。

wagtail 2.6.1 has requirement django-modelcluster<5.0,>=4.2, but you have django-modelcluster 5.0.

7. 最新版のパッケージのみを全て出力

 

$ pip list -u

 

 

8. インストール済みパッケージ全てを "パッケージ名"=="バージョン番号" 形式で表示

 

$ pip freeze

 

"requirements.txt" を作成するときに使用します。

$ pip freeze > requirements.txt

 

 

9. パッケージをアップグレード（バージョンアップ）

 

$ pip install -U <package-name>

 

 

10. pip 自体をアップグレード（バージョンアップ）

 

$ pip install -U pip

 

 

11. "requirements.txt" を使って一括インストール

 

$ pip install -r requirements.txt

 

 

12. パッケージ詳細内容を確認（バージョン、インストール場所など）

 

$ pip show <package-name>
