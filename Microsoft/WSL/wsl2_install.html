WSL インストール手順

 

  1. Windows で PowerShell または Windows コマンド プロンプトを管理者モードで開く
  2. wsl --install を実行：
    このコマンドにより、WSL を実行し、Linux の Ubuntu ディストリビューションをインストールするために必要な機能が有効になります。
    このコマンドは、WSL がまったくインストールされていない場合にのみ機能します。
  3. wsl --list --online を実行：
    使用可能なディストリビューションの一覧を出力します。

  4. wsl --install -d <DistroName> を実行：
    今回は例として Ubuntu-26.04 を使用します
    wsl --install -d Ubuntu-26.04

  5. Windows を再起動：
    ”システムを再起動する必要があります” を表示した場合、指示に従って Windows を再起動します。
  6. Linux 環境で使用するユーザー名、パスワードを登録：
    デフォルト状態であなたのユーザー名を表示。そのままでも良いですし、任意の名前へ変更しても OK です。
  7. 確認画面：
    下図画面を表示したら “n” を入力します。
    補足：
    これは、Ubuntu（Linux）初回起動時に表示されるデータ収集の同意確認メッセージです。Ubuntu があなたに対して
    「システム情報（ハード構成や利用状況）を匿名で Canonical に送ってもいいですか？」
    と確認しています。あなたがよければ "y" を入力しても良いですが、基本 "n" で良いでしょう。

  8. インストールを完了したら "exit" を入力して一旦終了
  9. ｢スタート｣ から ［Ubuntu 26.04］を起動

  10. Ubuntu 起動後、最新版へアップデート
    パスワードを聞かれるので、インストール時に設定したパスワードを入力 
    # パッケージリストを最新へ更新
sudo apt update
# 最新版へアップデート
sudo apt upgrade

 

 

参考： ダウンロード版によるインストール手順

 

  1. Ubuntu 公式サイト から .wsl ファイルをダウンロード
  2.  そのまま ダブルクリック

 

 これだけでインストールは完了です。初期ユーザー作成まで自動で進んで、すぐに使えます。

 

 

参考： WSL アンインストール手順

 

  1. インストール済み WSL を確認する
    削除対象のディストリビューション名を正確に把握します。
    PowerShell または Windows Terminal で実行
    •  wsl -l -v を実行
    •  一覧に表示される名前を確認（例： Ubuntu-26.04, Ubuntu-22.04, Ubuntu）

  2. 対象の Ubuntu を登録解除する
     WSL 内のファイル・設定・環境がすべて削除されます。
    •  wsl --unregister Ubuntu-26.04 のように実行
    • 名前は Step01 で確認したものを使う
    • 実行後、そのディストリビューションは WSL から完全に消える
  3. Windows 側の Ubuntu アプリをアンインストール
    Microsoft Store 版などダウンロード版をインストールしている場合は、
    設定 → アプリ → インストールされているアプリ
    を開いてアンインストールを実行します。

  4. WSL 全体を初期化したい場合（任意）
    Ubuntu だけでなく WSL 全体を削除する場合の手順です。
    •  wsl --shutdown を実行
    • 「Windows の機能」→「Linux 用 Windows サブシステム」のチェックを外す

    • PC を再起動
