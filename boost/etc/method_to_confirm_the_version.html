boost library のバージョン番号を確認する方法

boost library のバージョン番号を確認する方法です。

[プログラムソース "boost_version.cpp"] 
#include <iostream>            // cout, endl
#include <cstdlib>             // EXIT_SUCCESS
#include <boost/version.hpp>   // BOOST_VERSION

using namespace std;

int main()
{
    cout << "boost version = "
         << ( BOOST_VERSION / 100000 )    << "." 
         << ( BOOST_VERSION / 100 % 1000) << "." 
         << ( BOOST_VERSION % 100 )       << endl;

    return EXIT_SUCCESS ;
}

[実行結果] 

boost version = 1.40.0

OS: Ubuntu 10.04
作成：
    gcc 4.4.3
    boost 1.40.0
コンパイル： gcc -Wall -O3 boost_version.cpp -o boost_version.exe -lstdc++
