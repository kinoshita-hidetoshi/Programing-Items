vscode からローカルホームページを起動する方法

1. Live Server をインストール

  • vscode 左の拡張機能アイコンをクリック
  • 「Live Server」で検索
  • 作者が Ritwick Dey のものをインストール

 

2. ホームページ（HTML）があるフォルダを VSCode で開く

  • vscode メニュー → ［ファイル］→［フォルダーを開く］
  • index.html などが入ったフォルダを選択

 

3. HTML を右クリック→「Open with Live Server」

 または vscode 右下の 「Go Live」 をクリック。
すると自動でブラウザが開き、次の URL でページを表示します： 

http://127.0.0.1:5500/

 

 

🔥 Live Server を使うメリット

  • 自動リロード：保存するたびにブラウザが自動更新
  • JavaScript が正しく動作（file:// では動かない場合もある）
  • 設定不要で即使える

 

 

🧪 HTTPS でローカル起動したい場合（開発用）

Live Server は HTTP ですが、HTTPS が必要なら Python や Node のローカルサーバーを使う方法もあります。
